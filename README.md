Projeto de extensão, realizado pelos alunos do Instituto Mauá de Tecnologia em parceria com a ETEC Júlio de Mesquita. 

# 📝 Sobre o Projeto
Desenvolvimento de um jogo virtual, baseado nas funcionalidades do tradicional jogo de Dominó, no qual cada peça conterá conceitos de química inorgânica (ácidos, bases, sais e óxidos) e os jogadores terão que conectá-las (ex: fórmula <-> nome, fórmula <-> função) com o intuito reforçar o estudo do conteúdo, favorecendo uma abordagem lúdica que se distancia apenas da memorização. Com sistema de validação de encaixes em tempo real, o jogo registrará acertos, erros e tempo de partida que no final serão usados para gerar um relatório dos jogadores cadastrados. 

# 📌Público-alvo
Alunos do ensino médio, especialmente do 1° e 2° ano, da ETEC Júlio de Mesquita

# 🚀 Funcionalidades Principais
- Autenticação de usuários com login e senha criptografada
- Jogo de dominó com peças contendo conteúdos de química inorgânica
- Modo de jogo multiplayer local
- Modo de jogo com tabuleiro interativo
- Ranking de jogadores com a maior pontuação
- Opções de acessibilidade

# :computer: Tecnologias Utilizadas 
- `Java`
- `LibGDX`
- `MongoDB`
